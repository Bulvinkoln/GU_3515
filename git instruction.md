## Привет! Это инструкция по использованию депозиториев других пользователей ##

Для того, чтобы использовать депозитории других пользователей, вам нужно будет выполнить следующие шаги:

1. Зайти на аккаунт конкретного человека в GitHub и выбрать депозиторий с открытым доступом
2. После этого, вам нужно выполнить fork
3. Далее вам нужно вернуться в свой профиль, убедиться в том, что необходимый депозиторий переместился и скопировать ссылку, которая появится после нажатия на зеленую кнопку "Code" во вкладке HTTPS
4. В терминале VSC ввести команду "git clone (ссылка на депозиторий)". Убедитесь, что в конце нет тильды, иначе может появиться ошибка! 
5. Данные депозитория появятся в ранее открытой вами папке
6. Чтобы приступить к работе в самом депозитории, во вкладке "Файл" выберите "Открыть папку" и укажите папку с выбранным вами депозиторием
7. Ура, получилось!
