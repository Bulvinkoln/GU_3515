## Инструкция по работе с удаленным репозиторием

1. Создаем аккаунт на github.com
2. Создаем локальный репозиторий
3. Связываем локальный и удаленный репозиторий
4. Отправляем (push) локальный репозиторий в удаленный
5. Проводим изменения с другого компьютера
6. Выкачиваем (pull) актуальное состояние из удаленного репозитория
7. Делаем (fork) интересуещего репозитория
8. Делаем git clone для нашей версии этого репозитория
9. Создаем ветку с изменениями
10. Производим все изменения в этой ветке
11. Отправляем изменения на свой аккаунт (push)
12. pull request
